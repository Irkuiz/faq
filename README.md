# Горячие клавиши
### копировать: ctrl+c
### вставить:
ctrl+v
### сохранить:
ctrl+s
### вырезать:
ctrl+x
### отмена:
ctrl+z
# работа с git
### скачать проект:
```bash
git clone <URL>
``` 
### учесть все файлы:
```bash
git add .
```
### сохранить коммит:
```bash
git commit -m "..."
```
### отправка в облако:
```bash
git push
```
